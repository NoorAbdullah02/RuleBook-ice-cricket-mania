# RuleBook-ice-cricket-mania

A comprehensive rulebook website for Ice Cricket, showcasing the unique rules and gameplay of this frozen sport.

## Features

- Complete rulebook with detailed explanations
- Interactive gameplay guide
- Team and player information
- Equipment specifications
- Scoring system documentation

## How to Use

Open `index.html` in your browser to view the rulebook.
